RUTAS MARÍA JESÚS V1

App HTML/CSS/JS offline para registrar rutas de autobús.

FUNCIONES INCLUIDAS
- Añadir y editar rutas.
- Listado con búsqueda y filtros.
- Imprimir ruta individual.
- Imprimir listado filtrado.
- Copiar listado a Excel en formato TSV.
- Exportar/importar copia JSON.
- Copiar/pegar JSON manualmente.
- Configuración visual: título, temas, colores, letra, tamaño, transparencias y modo claro/oscuro/3D.
- ID único por dispositivo.
- Emparejamiento por ID.
- Sincronización preparada con Firebase Realtime Database mediante URL REST.
- Sincronización automática al arrancar si ya está configurado el emparejamiento.
- Resolución de conflictos: gana el registro con updatedAt más reciente.

IMPORTANTE SOBRE FIREBASE
Para que la sincronización funcione hay que crear un proyecto en Firebase Realtime Database y pegar la URL en la pantalla SINCRONIZACIÓN.
Ejemplo de URL:
https://TU-PROYECTO-default-rtdb.europe-west1.firebasedatabase.app

Durante pruebas, las reglas de Firebase deben permitir lectura/escritura al nodo usado por la app.
Después se recomienda protegerlas según el sistema definitivo que se quiera usar.
