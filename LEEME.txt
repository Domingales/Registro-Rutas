RUTAS MARIA JESUS V2

Cambios principales de esta versión:
- Añadida sincronización Firebase preparada para el proyecto registro-rutas-b3d66.
- URL de Realtime Database incluida directamente en el código.
- Ventana SINCRONIZACIÓN con ID de este dispositivo, ID del otro dispositivo, EMPAREJAR y SINCRONIZAR AHORA.
- Sincronización automática al abrir la app, al recuperar internet y cada 3 minutos.
- Sincronización bidireccional por dispositivo.
- Resolución de conflictos por fecha updatedAt: se conserva la versión modificada más recientemente.
- Borrado sincronizable: al borrar una ruta se guarda una marca interna de borrado para que no reaparezca en el otro dispositivo.
- Conserva localStorage para funcionamiento offline.

Firebase configurado:
https://registro-rutas-b3d66-default-rtdb.europe-west1.firebasedatabase.app

Reglas necesarias en Firebase Realtime Database:
{
  "rules": {
    ".read": true,
    ".write": true
  }
}

Uso recomendado:
1. Abrir la app en el primer dispositivo.
2. Entrar en SINCRONIZACIÓN y copiar el ID de este dispositivo.
3. Abrir la app en el segundo dispositivo.
4. Entrar en SINCRONIZACIÓN y copiar el ID de este segundo dispositivo.
5. En cada dispositivo, pegar el ID del otro dispositivo y pulsar EMPAREJAR.
6. Pulsar SINCRONIZAR AHORA en ambos dispositivos la primera vez.
7. A partir de entonces, la app sincroniza automáticamente al abrir, al recuperar internet y cada 3 minutos.
